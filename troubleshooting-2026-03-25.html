л””лІ„к№… нғҖмһ„лқјмқё

ERROR apprunner:ListServices к¶Ңн•ң кұ°л¶Җ

CD мӣҢнҒ¬н”ҢлЎңмҡ°м—җм„ң aws apprunner list-services нҳём¶ң мӢң AccessDeniedException

мӣҗмқё: ListServicesлҠ” нҠ№м • м„ңл№„мҠӨ ARNмңјлЎң м ңн•ң л¶Ҳк°Җ — Resource = "*" н•„мҡ”
FIX ListServicesлҘј лі„лҸ„ StatementлЎң л¶„лҰ¬ 
{
  "Sid": "AppRunnerList",
  "Action": ["apprunner:ListServices"],
  "Resource": "*"
}
ERROR Terraform state S3 м ‘к·ј л¶Ҳк°Җ

Infrastructure мӣҢнҒ¬н”ҢлЎңмҡ°м—җм„ң s3:GetObject on terraform state лІ„нӮ· кұ°л¶Җ

л¬ём ң: лӢӯкіј лӢ¬кұҖ — deployerм—җ S3 к¶Ңн•ңмқ„ м¶”к°Җн•ҳл Өл©ҙ terraformмқҙ н•„мҡ”н•ҳм§Җл§Ң, terraformмқ„ мӢӨн–үн•ҳл Өл©ҙ S3 к¶Ңн•ңмқҙ н•„мҡ”
FIX лЎңм»¬м—җм„ң terraform applyлЎң bootstrap

S3, DynamoDB(state lock) к¶Ңн•ңмқ„ лЎңм»¬ terraformмңјлЎң м§Ғм ‘ м Ғмҡ©

ERROR IAM мқёлқјмқё м •мұ… 2KB м ңн•ң мҙҲкіј 
LimitExceeded: Maximum policy size of 2048 bytes exceeded for user
FIX мқёлқјмқё м •мұ… → Managed PolicyлЎң м „нҷҳ

aws_iam_user_policy (2KB) → aws_iam_policy + aws_iam_user_policy_attachment (6KB)

ERROR ssm:DescribeParameters к¶Ңн•ң кұ°л¶Җ

Terraformмқҙ SSM Parameter metadata мЎ°нҡҢ мӢң н•„мҡ”н•ң к¶Ңн•ң лҲ„лқҪ

FIX SSMDescribe Statement м¶”к°Җ (Resource: *)
ERROR App Runner м•ұ мӢңмһ‘ мӢӨнҢЁ - DataSource м„Өм • мҳӨлҘҳ

ParameterStorePropertySource (EnvironmentPostProcessor)мқҳ лЎңк·ёк°Җ м „нҳҖ м¶ңл Ҙлҗҳм§Җ м•ҠмқҢ

мқҳмӢ¬ 1: AWS_REGION нҷҳкІҪліҖмҲҳ лҲ„лқҪ → м¶”к°Җн–Ҳм§Җл§Ң н•ҙкІ° м•Ҳ лҗЁ
мқҳмӢ¬ 2: VPC Endpoint н•„мҡ”? → NAT Instance кІҪмң лЎң к°ҖлҠҘн•ҙм•ј н•Ё
INSIGHT EnvironmentPostProcessor мһҗмІҙк°Җ нҳём¶ңлҗҳм§Җ м•ҠмқҢ

лЎңк№… мҪ”л“ңлҘј м¶”к°Җн–Ҳм§Җл§Ң Loading SSM parameters... лЎңк·ёк°Җ м—ҶмқҢ — нҒҙлһҳмҠӨ лЎңл”© мһҗмІҙк°Җ мӢӨнҢЁ

INSIGHT bootJar лӮҙл¶Җ л¶„м„қ 
$ unzip -l *.jar | grep "aws.*ssm"
(кІ°кіј м—ҶмқҢ!)

$ unzip -l *.jar | grep "SClass-Infrastructure"
BOOT-INF/lib/SClass-Infrastructure-0.0.1-SNAPSHOT-plain.jar

Infrastructure JARмқҖ нҸ¬н•Ёлҗҳм–ҙ мһҲм§Җл§Ң, AWS SSM SDK JARмқҙ bootJarм—җ м—ҶмқҢ

ROOT CAUSE FIX implementation → api мҠӨмҪ”н”„ ліҖкІҪ

SClass-Infrastructure/build.gradle.ktsм—җм„ң мқҳмЎҙм„ұ мҠӨмҪ”н”„ ліҖкІҪ

// Before (м „мқҙ мқҳмЎҙм„ұ м „нҢҢ м•Ҳ лҗЁ)
implementation("software.amazon.awssdk:ssm")

// After (bootJarм—җ нҸ¬н•ЁлҗЁ)
api("software.amazon.awssdk:ssm")

к·јліё мӣҗмқё

Gradle л©ҖнӢ°лӘЁл“Ҳм—җм„ң implementation мҠӨмҪ”н”„лҠ” м „мқҙ мқҳмЎҙм„ұмқ„ м „нҢҢн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

SClass-Infrastructureк°Җ implementationмңјлЎң AWS SDKлҘј м„ м–ён•ҳл©ҙ, мқҙлҘј мқҳмЎҙн•ҳлҠ” SClass-Api-Supportersмқҳ bootJarм—җ AWS SDK JARмқҙ нҸ¬н•Ёлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.

кІ°кіј: ParameterStorePropertySourceк°Җ SsmClient нҒҙлһҳмҠӨлҘј м°ҫм§Җ лӘ»н•ҙ EnvironmentPostProcessorк°Җ мЎ°мҡ©нһҲ мӢӨнҢЁ → SSM к°’ лҜёмЈјмһ… → DataSource м„Өм • мӢӨнҢЁ

көҗнӣҲ

мЈјм ң көҗнӣҲ
implementation vs api лқјмқҙлёҢлҹ¬лҰ¬ лӘЁл“Ҳм—җм„ң лӢӨлҘё лӘЁл“Ҳмқҙ лҹ°нғҖмһ„м—җ мӮ¬мҡ©н•ҙм•ј н•ҳлҠ” мқҳмЎҙм„ұмқҖ л°ҳл“ңмӢң apiлЎң м„ м–ё
EnvironmentPostProcessor нҒҙлһҳмҠӨ лЎңл”© мӢӨнҢЁ мӢң лЎңк·ё м—Ҷмқҙ л¬ҙмӢңлҗЁ — л””лІ„к№…мқҙ л§Өмҡ° м–ҙл ӨмӣҖ
